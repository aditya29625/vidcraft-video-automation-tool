# VidCraft Frontend 🎨

The frontend for VidCraft AI, built with React, Vite, and Tailwind CSS.

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up environment variables:
   Create a `.env` file:
   ```env
   VITE_BACKEND_URL=http://localhost:3000
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Features
- User-friendly interface for content generation.
- Dynamic theme with glassmorphism effects.
- Real-time AI content display.
